# Voice_Based_file_authentication
Python mini project for voice-based file authentication. 

## Project Overview
The system stores a fixed authorized voice and compares it with a live voice input.
Access to a protected file is granted only if both voices match.

##  Technologies Used
- Python
- Librosa
- SoundDevice
- NumPy

## How to Run
1. Run "register_voice.py" once to store the fixed voice
2. Run "auth_voice.py" to authenticate and access the file
